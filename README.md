## üöÄ Getting Started

### Prerequisites
- Node.js 16+
- MongoDB 4.4+
- Yarn/npm

### Installation
```bash
# Clone repository
git clone https://github.com/yourusername/mangaverse.git
cd mangaverse

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your credentials

# Start development server
nodemon / node server.js
